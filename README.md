# web-prog-rgr

Клиентская игра на внимание и скорость реакции для расчетно-графического задания по JavaScript.

## Запуск

1. Откройте `index.html` в браузере.
2. Выберите размер таблицы.
3. Нажмите `Старт` и кликайте по числам в порядке возрастания.

## Архитектура

- `js/domain/` — правила предметной области: генерация поля, стили чисел, состояние партии.
- `js/app/` — координация сценария игры и таймера.
- `js/ui/` — рендер интерфейса и работа с DOM через jQuery.
- `js/main.js` — точка входа.

## Реализовано

- квадратная таблица с числами от `1` до `N` в случайном порядке;
- разные цвет и размер для каждого числа;
- последовательный выбор чисел;
- ограничение по времени, пропорциональное числу ячеек;
- изменение фона ячейки при правильном выборе;
- использование DOM API и jQuery.
